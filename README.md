# Worms

## Installation
- Run the Install_vcpkg.bat
- Go in the Build folder.
- Launch WormsApp.sln
- Set WormsApp as the Default project.
- Build on debug or release mode.
- You can directly play the .exe in the build/WormsApp/(Debug or Release)/WormsApp.exe
